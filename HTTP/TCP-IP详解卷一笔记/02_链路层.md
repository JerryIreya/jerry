# 引言

从图1-4中可以看出，在TCP/IP协议族中，链路层主要有三个目的：

1. 为IP模块发送和接受IP数据报
2. 为ARP模块发送ARP请求和接收ARP应答
3. 为RARP发送RARP请求和接收RARP应答

TCP/IP支持多种不同的链路层协议，这取决于网络所使用的硬件，如以太网、令牌环网等。
